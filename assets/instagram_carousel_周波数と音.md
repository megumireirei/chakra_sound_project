## 【スライド 2 / 8】周波数とは？

### 見出し
```
「周波数」ってそもそも何？
```

### 本文
```
音は"空気の振動"。

その振動の速さ（1秒間に何回揺れるか）を
Hz（ヘルツ）で表したものが「周波数」。

人の声も、音楽も、自然の音も——
すべては周波数の集合体。

そして、あなたの身体も。
```

### 引用テキスト（目立たせる）
```
「すべての物質は振動している」
— ニコラ・テスラ
```

### デザインメモ
- 波形グラフィックをさりげなく背景に
- シンプルに白テキスト×ダークバック

## 【スライド 5 / 8】432Hz vs 440Hz

### 見出し
```
あなたが今聴いている音楽は
"ずれている"かもしれない
```

### 本文
```
現代の音楽のほとんどは 440 Hz で
チューニングされている。

でも、それが標準化されたのは1953年。
それ以前の多くの楽器や音楽は
432 Hz で演奏されていた。

432 Hzは、自然界の法則（フィボナッチ数列・黄金比）
と共鳴する周波数とも言われている。

同じ曲を432Hzで聴くだけで——
なぜか身体が、すーっと落ち着く。
```

### 比較テキスト（視覚的に）
```
440 Hz ── 現代標準・やや刺激的
432 Hz ── 自然共鳴・身体に馴染む
```

### デザインメモ
- 二分割デザイン（左440、右432）
- 右側をほんのり暖かい色に
