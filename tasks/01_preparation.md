# STEP 1: 準備

- [x] note.com でアカウント登録済み

## 音源制作環境の確認

- [ ] DAWソフトを用意する
  - 無料: Audacity、GarageBand（Mac）
  - 有料: Logic Pro、Ableton Live、FL Studio
- [ ] ヘッドフォンまたはモニタースピーカーを用意
- [ ] 音源の出力形式を決める（推奨: MP3 320kbps または WAV 44.1kHz）

## コンセプト決め

- [ ] ターゲットを決める（瞑想初心者 / 上級者 / ヨガ実践者 など）
- [ ] 音源のスタイルを決める
  - 純粋な周波数トーン（シンプル）
  - 自然音 + 周波数（川の音、雨音など）
  - シンギングボール + 周波数
  - バイノーラルビート
- [ ] 1音源あたりの長さを決める（目安: 10〜30分）
