# STEP 3: Note出品

## 記事作成の準備

- [ ] サムネイル画像を作成（推奨サイズ: 1280×670px）
  - ツール: Canva（無料）が使いやすい
  - チャクラのシンボル・色・周波数を入れたデザイン
- [ ] 各チャクラの説明文を書く（200〜400字目安）
- [ ] 使い方・注意事項テキストを作成

## Note有料記事の投稿手順

- [ ] Noteにログイン（https://note.com/megumirei）
- [ ] 「投稿」→「テキスト」を選択
- [ ] タイトルを記入（例: 「【639Hz】ハートチャクラ ヒーリング音源｜愛と癒しの周波数」）
- [ ] 本文を記入（下記テンプレート参照）
- [ ] 音声ファイルをアップロード（Note直接アップ または ダウンロードリンクを記載）
- [ ] サムネイルを設定
- [ ] 価格を設定して「有料公開」で投稿

## 記事本文テンプレート

```
## この音源について
〇〇チャクラ（○○Hz）に対応したヒーリング音源です。
[チャクラの説明・効果を記載]

## こんな方におすすめ
- [ターゲット1]
- [ターゲット2]
- [ターゲット3]

## 収録内容
- 収録時間: 約〇〇分
- 形式: MP3（320kbps）
- 周波数: ○○○Hz

## 使い方
1. ヘッドフォンまたはスピーカーで再生してください
2. 目を閉じ、リラックスした姿勢で聴きます
3. 対応するチャクラの場所に意識を向けてください

## 注意事項
- 運転中や機械操作中はご使用をお控えください
- 購入後の返金はお受けできません
- 個人利用のみとし、転載・再配布はご遠慮ください
```

## 商品ラインナップ構成

- [x] 無料記事を3本投稿済み（集客・信頼獲得）
  - スピリチュアルじゃない、"周波数"のはなし
  - 「動けない」は意志の問題じゃない。（1分間ミニ音源付き）
