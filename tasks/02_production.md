# STEP 2: 音源制作

## 制作するチャクラ音源（全7種）

| 優先度 | チャクラ | 周波数 | 対応テーマ | 状態 |
|--------|---------|--------|-----------|------|
| 1 | 第4チャクラ（ハート） | 639 Hz | 愛・癒し | 未制作 |
| 2 | 第1チャクラ（ルート） | 396 Hz | 安心・グラウンディング | 未制作 |
| 3 | 第3チャクラ（ソーラープレクサス） | 528 Hz | 自信・活力 | 未制作 |
| 4 | 第2チャクラ（サクラル） | 417 Hz | 創造性・感情解放 | 未制作 |
| 5 | 第5チャクラ（スロート） | 741 Hz | 表現・コミュニケーション | 未制作 |
| 6 | 第6チャクラ（サードアイ） | 852 Hz | 直感・洞察 | 未制作 |
| 7 | 第7チャクラ（クラウン） | 963 Hz | 宇宙との繋がり | 未制作 |

> ハートチャクラ（639Hz）から始めると「癒し・愛」というテーマで売れやすい

## 各音源の制作手順

- [ ] 対応周波数のベーストーンを生成
- [ ] バックグラウンドサウンドを選ぶ（自然音・シンギングボールなど）
- [ ] ミックス・音量バランスを調整
- [ ] イントロ・アウトロにフェードイン/アウトを入れる
- [ ] 試聴して違和感がないか確認
- [ ] MP3（320kbps）とWAVの両形式で書き出し
- [ ] ファイル名をわかりやすく命名
  - 例: `chakra_04_heart_639hz.mp3`

## 品質チェック

- [ ] ノイズ・雑音がないか確認
- [ ] 音量が適切か（ヘッドフォンで聴いて耳が痛くないレベル）
- [ ] 指定の周波数が正しく含まれているか確認
- [ ] 最低10分以上の長さがあるか

## ファイル管理

```
chakra_sound_project/
└── sounds/
    ├── masters/        # 編集用マスターファイル
    ├── exports/        # 書き出し済みMP3/WAV
    └── samples/        # 無料サンプル用（短縮版）
```
