# STEP 4: 集客・マーケティング

## SNSアカウント整備

- [ ] X（Twitter）プロフィールにNoteリンクを追加（https://note.com/megumirei）
- [ ] Instagramプロフィールにリンクを追加（https://note.com/megumirei）
- [ ] 各SNSのプロフィールに「チャクラ音源 / ヒーリング」と明記

## 発信コンテンツ計画

### 無料で価値提供（信頼を積む）
- [ ] チャクラの基礎知識を発信（各チャクラの意味・整え方）
- [ ] 短い音源サンプル（30秒〜1分）をSNSに投稿
- [ ] 「なぜチャクラ音源を作ったか」のストーリーを発信

### 購買につながる発信
- [ ] Note記事投稿時にSNSで告知
- [ ] 購入者の声（レビュー）を共有
- [ ] 期間限定割引・セットキャンペーンを告知
