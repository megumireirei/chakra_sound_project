# チャクラ音源プロジェクト

チャクラの各エネルギーセンターに対応した音源を制作し、Noteにて販売するプロジェクトです。

## プロジェクト概要

7つのチャクラそれぞれに対応したヒーリング音源を制作・販売します。
音楽の力で心身のバランスを整え、より豊かな日常をサポートすることを目的としています。

## 音源ラインナップ（予定）

| チャクラ | 対応部位 | 周波数 |
|---------|---------|--------|
| 第1チャクラ（ルート） | 安心・安定 | 396 Hz |
| 第2チャクラ（サクラル） | 創造性・感情 | 417 Hz |
| 第3チャクラ（ソーラープレクサス） | 自信・意志力 | 528 Hz |
| 第4チャクラ（ハート） | 愛・癒し | 639 Hz |
| 第5チャクラ（スロート） | 表現・コミュニケーション | 741 Hz |
| 第6チャクラ（サードアイ） | 直感・洞察力 | 852 Hz |
| 第7チャクラ（クラウン） | 宇宙との繋がり | 963 Hz |

## 販売プラットフォーム

- **Note**: チャクラ音源の単品販売・セット販売

## 使用方法

1. Noteにて音源を購入・ダウンロード
2. ヘッドフォンまたはスピーカーで再生
3. リラックスした状態で目を閉じ、対応するチャクラを意識しながら聴く

## ライセンス

本音源の無断転載・再配布はご遠慮ください。個人利用に限りご使用いただけます。
