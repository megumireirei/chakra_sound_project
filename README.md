# BLOOMING SOUND

**感覚を整え、現実を変える音。**

周波数デザイナー REI（れいちゃん）が手がける、ニューロサウンドブランド。
脳や神経に直接アプローチする音源を通じて、もっと自然に、もっと軽やかな自己実現をサポートします。

---

## ブランドコンセプト

音は、ただ聴くものじゃない。内側を"開く"もの。

思考では届かない深層へ。神経回路に直接アプローチする**ニューロサウンド**は、
脳・神経・感覚へ働きかけるために設計された音源です。

ただ癒すだけではなく、"いつもの思考パターン"や"無意識のノイズ"へアプローチします。

### BLOOM — 内側を咲かせる

音を浴びることで、眠っていた感覚が少しずつ開いていく。

- 自分を感じやすくなる
- 感覚がクリアになる
- 本来のエネルギーを思い出す

「変わる」のではなく、*"本来のあなたが、動き出す"* 感覚。

### CONNECT — 音で、心と脳を繋ぐ

思考だけでは進めない時。頑張っているのに噛み合わない時。
音が、バラバラになった感覚を繋ぎ直していく。
心・脳・身体・感覚。その全体が整い始める。
